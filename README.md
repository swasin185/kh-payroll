## **ลักษณะงานที่รับผิดชอบ**

### **ด้าน Data / Database**
- วิเคราะห์และดึงข้อมูลด้วย SQL  
- ใช้งาน MariaDB และ SkySQL  
- จัดการฐานข้อมูลด้วย DBeaver  

### **ด้าน Development / Web Application**
- พัฒนาเว็บแอปด้วย Nuxt.js, Node.js และ Typescript  
- ใช้งาน GitHub เพื่อจัดการเวอร์ชันโค้ด  

### **ด้าน Report**
- ออกแบบรายงานด้วย JasperStudio (Java)  
- สร้างกราฟและ Visualization ด้วย NuxtChart  

### **ด้าน System (Linux)**
- ใช้งานและดูแลระบบพื้นฐานบน Linux (Debian / Ubuntu)

### **ด้าน IT Support**
- ดูแล ตรวจสอบ และแก้ไขอุปกรณ์ IT เช่น คอมพิวเตอร์ ปริ้นเตอร์ อุปกรณ์เครือข่าย  
- ติดตั้งซอฟต์แวร์พื้นฐานและสนับสนุนผู้ใช้งานภายในองค์กร

# **Project-Based Learning (ระหว่างฝึกงาน)**

ผู้ฝึกงานจะได้ทำงานในรูปแบบ **Project จริงของบริษัท** โดยเน้นการลงมือทำและแก้ปัญหาจริง เช่น:

- **สร้าง Web Dashboard** ด้วย Nuxt.js + NuxtChart  
- **ทำระบบดึงข้อมูล** จาก MariaDB / SkySQL เพื่อนำไปใช้งาน  
- **ออกแบบรายงาน** ด้วย JasperStudio สำหรับแผนกต่าง ๆ  
- **พัฒนา API เบื้องต้น** เพื่อเชื่อมระบบภายใน  
- **ปรับแต่งเซิร์ฟเวอร์ Linux** สำหรับงานเฉพาะด้าน  
- **งาน IT Support ที่เกิดขึ้นจริงในองค์กร**

(หัวข้องานอาจปรับตามความเหมาะสมของแต่ละแผนกและทักษะของผู้ฝึกงาน)
